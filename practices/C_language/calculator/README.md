====================================================
=               How it works                       =
====================================================
The program ask for 3 input from user
	1. first value
	2. operator ('+', '-', '/',  '*')
	3. Second value
Note the 3 inputs must be enter after the prompt in the rectancle in this form
			3 * 2
here after the value will be displayed
